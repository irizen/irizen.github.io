Remarker

Remarker is a tool for MacOS that will be useful to those who work with textual information, and is an advanced way to create comments, which are called remarks in the app, on portions of text. It differs from most existing note taking apps in that it allows you to comment on overlapping areas of text. To do this, the app provides colored frames around the text being commented, which outline remarks. In addition, remarks can be assigned categories, which are stored in their own arbitrary hierarchy, allowing you to create your own knowledge base from your saved quotes.

Otherwise, Remarker is simply a platform for collecting quotes. It allows advanced searches for both quotes and remarks. It also allows you to store the database in iCloud, which is useful as a backup tool as well as a means of sharing the database between your devices.

Sphere-like color picker

Classic color pickers are a bit boring and abstract. People perceive simple metaphors from the real world better. I suggest using a color picker in the form of an elegant sphere (a globe) on the surface of which the HSL color space is UV-mapped with the U-axis for hue, the V-axis for lightness, and the shrinking of the sphere represents saturation:

For a live demo, see the free CVAssistant iOS app.

Playmap-like navigation

For the presentation of groups of objects, the classic tabular navigation does not really match the trends of graphical UIs. By adding a little skeuomorphism we can really revive the navigation. Why not use the metaphor of playing cards?

For a live demo, see the free CVAssistant iOS app.

Interactive Chart

Presenting statistical data to the user for effective examination and extraction of useful information is not a trivial task. It's crucial to be able to go through all the data and zoom in on an interesting range. An intuitive interface is very important for presenting abstract information and graphics and animations are very effective. I developed this component as part of a project.
